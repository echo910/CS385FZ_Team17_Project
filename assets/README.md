# Assets 文件夹说明

## 需要添加的图片文件

请将以下图片文件放入此文件夹：

### Hero Cards（正方形图片，160x160 推荐）

1. **hero1.png** - 光头艺术家肖像
   - 描述：灰色背景，深色外套，银色耳环
   - 用途：Heartbeat Radar 卡片

2. **hero2.png** - 白色毛衣艺术家
   - 描述：深灰色背景，米白色毛衣，卷发
   - 用途：Similar Artists 卡片

3. **hero3.png** - 粉色插画人物
   - 描述：绿色背景，粉色人物，心形元素
   - 用途：Daily Mix 卡片

### Album 封面（专辑图片）

1. **album1.png** - 方大同专辑
   - 描述："THIS LOVE, KHALIL FONG" / "愛愛愛 方大同"
   - 条纹衬衫，太阳镜

2. **album2.png** - 王力宏专辑
   - 描述："心中的日月" / "Shangri-La"
   - 深色连帽衫，白色T恤

3. **album3.png** - 其他专辑封面

### Artist 头像（圆形显示）

1. **artist1.png** - 艺术家头像 1
2. **artist2.png** - 艺术家头像 2
3. **artist3.png** - 艺术家头像 3

## 图片规格要求

- **Hero Cards**: 160x160 像素（正方形）
- **Album**: 120x120 或 140x140 像素（正方形）
- **Artist**: 110x110 像素（将显示为圆形）
- **格式**: PNG 或 JPG
- **质量**: 清晰，适合移动端显示

## 注意事项

- 所有文件名必须与上述名称完全一致（区分大小写）
- 如果图片文件不存在，应用会报错
- 建议使用 PNG 格式以获得更好的透明度支持
